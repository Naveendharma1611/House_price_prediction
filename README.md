# House Price Prediction using Machine Learning

##  Project Overview

This project aims to predict house prices using Machine Learning algorithms based on features such as area, number of bedrooms, location, and other property-related attributes.

It is a **Supervised Learning – Regression** problem where the target variable is continuous (house price).

##  Objectives

* Perform data preprocessing and cleaning
* Apply regression algorithms
* Evaluate model performance
* Improve prediction accuracy
* Build an end-to-end ML pipeline

## 📊 Dataset

The dataset contains features like:

* Area (sqft)
* Bedrooms
* Bathrooms
* Location
* Age of property
* Parking availability
* Price (Target variable)

Dataset Source:

* Kaggle House Price Dataset / Custom dataset


##  Technologies Used

* Python
* Pandas

* Matplotlib / Seaborn


##  Machine Learning Workflow

1. Data Collection
2. Data Cleaning

5. Model Training
6. Model Evaluation
7. Prediction


##  Models Used

* Linear Regression
* Ridge / Lasso Regression
* Decision Tree Regressor
* Random Forest Regressor

Best performing model: **Random Forest Regressor** (based on R² score)


## 📈 Evaluation Metrics

* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## 📊 Results

* R² Score: 0.85+
* Low MSE value
* Good generalization on test data
